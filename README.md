# ColorCpp
C++20 ColorSpace Module

## About
"ColorCpp" is color spaces library by C++20 module.

We support the following color spaces.

- A98
- CMY
- CMYK
- DisplayP3
- Hex
- HSI
- HSL
- HSV
- HunterLab
- HWB
- ITP
- Lab
- LCH
- LRGB
- Luv
- OkLab
- OkLCH
- Prophoto
- Rec2020
- RGB
- RGB24
- RGBA
- RGBA32
- RYB
- XyY
- XYZ
- YIQ

and the color difference.

- CIE76
- CIE94
- CIE2000
- CMC
- Euclidean
